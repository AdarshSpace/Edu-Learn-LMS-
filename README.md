# Edu Learn (LMS)

Edu Learn is a modern full-stack learning platform designed for educators to create, manage, and sell courses, and for students to learn through structured video lessons, study materials, and AI-powered doubt solving.

Built to make online learning more interactive, structured, and accessible.

---

## 🚀 Highlights

- Structured course learning (Course → Topic → Subtopic → Videos)
- Secure authentication and role-based access
- Course purchase and payment management
- AI-powered Q&A from uploaded PDF notes
- Secure media uploads and content delivery
- Scalable architecture for future growth

---
